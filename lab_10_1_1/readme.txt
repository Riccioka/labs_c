Условия первой задачи.
На вход через командную строку подается текстовой файл со странами в формате "название страны, ее столица, количество жителей" по ключу, переданному через командную строку (2-ым параметром):

на работу с одним элементом списка:
pf - вызывает функцию pop_front(), возвращающую указатель на данные из элемента, который расположен в "голове" списка, сам элемент удаляется - ./app.exe pf INPUT_FILE OUTPUT_FILE
pb - вызывает функцию pop_back(), возвращающую указатель на данные из элемента, который расположен в "хвосте" списка, сам элемент удаляется - ./app.exe pb INPUT_FILE OUTPUT_FILE

на работу с целым списком:
a - вызывает функцию append(), добавляющую один список в конец другого, при этом первый оказывается пустым - ./app.exe a INPUT_1_FILE INPUT_2_FILE OUTPUT_FILE

сортировка списка:
s - сортировка списка по названию страны и вставка элемента - ./app.exe s INPUT_FILE OUTPUT_FILE

Вид элемента в файле:

typedef struct country
{
    char *name_country;
    char *name_capital;
    long population;
} country_t;
